## German Cases: Akkusativ & Dativ - Complete Reference Guide
**(জার্মান কারক: আক্কুসাটিভ ও ডাটিভ - সম্পূর্ণ নির্দেশিকা)**

---

### Part 1: Quick Reference Tables (দ্রুত দেখার সারণী)

#### Akkusativ Prepositions (পুরো আক্কুসাটিভ)
These prepositions are ALWAYS followed by Akkusativ.
(এই অব্যয়গুলি সর্বদা আক্কুসাটিভ নিয়ে আসে।)

| Preposition | Meaning | Example |
| :--- | :--- | :--- |
| **für** | for | Das Geschenk ist für **den** Vater. (The gift is for the father.) |
| **um** | around | Wir sitzen um **den** Tisch. (We sit around the table.) |
| **ohne** | without | Er geht ohne **einen** Mantel. (He goes without a coat.) |
| **durch** | through | Wir gehen durch **den** Park. (We walk through the park.) |
| **gegen** | against | Das Auto fuhr gegen **den** Baum. (The car hit against the tree.) |
| **bis** | until | Bis **nächsten** Montag! (Until next Monday!) |
| **entlang** | along | Wir gehen **den** Fluss entlang. (We walk along the river.) |

---

#### Dativ Prepositions (পুরো ডাটিভ)
These prepositions are ALWAYS followed by Dativ.
(এই অব্যয়গুলি সর্বদা ডাটিভ নিয়ে আসে।)

| Preposition | Meaning | Example |
| :--- | :--- | :--- |
| **aus** | out of/from | Ich komme aus **der** Türkei. (I come from Turkey.) |
| **bei** | at/near | Ich wohne bei **meiner** Schwester. (I live at my sister's place.) |
| **mit** | with | Wir fahren mit **dem** Auto. (We go by car.) |
| **nach** | after/to | Nach **der** Schule gehe ich nach Hause. (After school I go home.) |
| **seit** | since/for | Seit **einem** Jahr lerne ich Deutsch. (I've been learning German for a year.) |
| **von** | from/of | Das ist ein Geschenk von **meiner** Mutter. (That's a gift from my mother.) |
| **zu** | to | Ich gehe zu **meinem** Freund. (I'm going to my friend's place.) |
| **ab** | from (time) | Ab **nächster** Woche. (From next week.) |
| **außer** | except | Alle außer **mir** waren da. (Everyone except me was there.) |
| **gegenüber** | opposite | Das Hotel liegt gegenüber **dem** Bahnhof. (The hotel is opposite the station.) |

---

#### Two-Way Prepositions (Wechselpräpositionen - মিশ্র অব্যয়)
These prepositions can take **Akkusativ** OR **Dativ** depending on the meaning!
(এই অব্যয়গুলি অর্থের উপর নির্ভর করে আক্কুসাটিভ অথবা ডাটিভ নিতে পারে!)

| Preposition | Meaning |
| :--- | :--- |
| **an** | on (vertical), at |
| **in** | in, into |
| **auf** | on (horizontal), onto |
| **vor** | in front of, ago |
| **über** | above, over, about |
| **unter** | under, among |
| **neben** | next to, beside |
| **hinter** | behind |
| **zwischen** | between |

**The Big Rule:**
- **Akkusativ** = **Movement** / Change of location (Wohin? - Where to?)
  (নড়াচড়া / স্থান পরিবর্তন - কোথায় যাচ্ছে?)
- **Dativ** = **Location** / Static position (Wo? - Where?)
  (অবস্থান / স্থির অবস্থা - কোথায় আছে?)

**Examples:**
- **Akkusativ (Movement):** Ich lege das Buch **auf den Tisch**. (I put the book **onto the table**.)
- **Dativ (Location):** Das Buch liegt **auf dem Tisch**. (The book is lying **on the table**.)

### Part 2: Mr. MN Shortcut for Dativ
**(ডাটিভের জন্য মি. এমএন শর্টকাট)**

This is a brilliant shortcut you mentioned! Here's how it works:
(এটি একটি চমৎকার শর্টকাট! এটি কীভাবে কাজ করে তা এখানে দেওয়া হল:)

**Mr. MN helps you remember how articles change in Dativ:**
**(মি. এমএন আপনাকে ডাটিভে আর্টিকেলের পরিবর্তন মনে রাখতে সাহায্য করে।)**

| Gender | Nominativ (The) | Mr. MN Rule | Dativ (The) |
| :--- | :--- | :--- | :--- |
| **M**asculine (der) | **der** | M -> **M** | **dem** |
| Fe**M**inine (die) | **die** | (Feminine is special: add **r**) | **der** |
| Neute**N** (das) | **das** | N -> **M** | **dem** |
| Plural (die) | **die** | Plural -> **N** | **den** + **n** (to noun) |

**Detailed Explanation (বিস্তারিত ব্যাখ্যা):**

1.  **M**asculine (der): **M** changes to **M** = **dem**
    - *der* Mann → *dem* Mann
    - *ein* Mann → *einem* Mann

2.  **Feminine (die)**: Add **r** at the end
    - *die* Frau → *der* Frau
    - *eine* Frau → *einer* Frau

3.  **Neuter** (das): **N** changes to **M** = **dem**
    - *das* Kind → *dem* Kind
    - *ein* Kind → *einem* Kind

4.  **Plural** (die): Add **N**
    - *die* Kinder → *den* Kinder**n**
    - (No indefinite article in plural)

### Part 3: How to Detect Akkusativ
**(আক্কুসাটিভ কীভাবে চিনবেন)**

You mentioned three ways to detect Akkusativ. Let's expand on each:

#### 1. WAS Object / Object Type (কী/কাকে - বস্তুর ধরন)
Ask the question: **Wen?** (Whom?) or **Was?** (What?)
(প্রশ্ন করুন: **কাকে?** বা **কী?**)

- If you can ask "Wen oder was?" after the verb, it's likely Akkusativ.
- **Example:** Ich sehe **den Mann**. (I see the man.)
    - Question: Wen sehe ich? (Whom do I see?) → **Den Mann** (Akkusativ)

**Bangla Test:** Does it answer "কে/কীকে"?
- আমি লোকটিকে দেখি। (কে? → লোকটিকে = Akkusativ)

#### 2. Preposition (অব্যয়)
Look for Akkusativ prepositions (für, um, ohne, durch, gegen, bis, entlang).
(আক্কুসাটিভ অব্যয়গুলি খুঁজুন।)

- **Example:** Das ist **für meinen Bruder**. (That is for my brother.)
    - *für* always takes Akkusativ → **meinen** Bruder

#### 3. Verb (ক্রিয়া)
Some verbs are always followed by Akkusativ (transitive verbs).
(কিছু ক্রিয়া সর্বদা আক্কুসাটিভ নিয়ে আসে।)

**Common Akkusativ Verbs:**
| Verb | Meaning | Example |
| :--- | :--- | :--- |
| **haben** | to have | Ich habe **einen** Hund. (I have a dog.) |
| **sehen** | to see | Ich sehe **den** Lehrer. (I see the teacher.) |
| **kaufen** | to buy | Er kauft **ein** Auto. (He buys a car.) |
| **lieben** | to love | Ich liebe **dich**. (I love you.) |
| **essen** | to eat | Wir essen **einen** Apfel. (We eat an apple.) |
| **trinken** | to drink | Sie trinkt **den** Kaffee. (She drinks the coffee.) |
| **brauchen** | to need | Ich brauche **einen** Bleistift. (I need a pencil.) |

### Part 4: How to Detect Dativ
**(ডাটিভ কীভাবে চিনবেন)**

#### 1. WEM Object / Object Type (কাকে/কার জন্য)
Ask the question: **Wem?** (To whom?/For whom?)
(প্রশ্ন করুন: **কাকে?** বা **কার জন্য?**)

- If you can ask "Wem?" after the verb, it's likely Dativ.
- **Example:** Ich helfe **dem Mann**. (I help the man.)
    - Question: Wem helfe ich? (Whom do I help?) → **Dem Mann** (Dativ)

**Bangla Test:** Does it answer "কাকে?" in the sense of "to/for whom"?
- আমি লোকটিকে সাহায্য করি। (কাকে? → লোকটিকে, but here it's indirect object)

#### 2. Preposition (অব্যয়)
Look for Dativ prepositions (aus, bei, mit, nach, seit, von, zu, ab, außer, gegenüber).
(ডাটিভ অব্যয়গুলি খুঁজুন।)

- **Example:** Ich wohne **bei meiner** Tante. (I live at my aunt's place.)
    - *bei* always takes Dativ → **meiner** Tante

#### 3. Verb (ক্রিয়া)
Some verbs are always followed by Dativ! These are special verbs.
(কিছু ক্রিয়া সর্বদা ডাটিভ নিয়ে আসে! এগুলি বিশেষ ক্রিয়া।)

**Common Dativ Verbs:**
| Verb | Meaning | Example |
| :--- | :--- | :--- |
| **helfen** | to help | Ich helfe **dem** Freund. (I help the friend.) |
| **danken** | to thank | Ich danke **der** Frau. (I thank the woman.) |
| **gefallen** | to please/like | Das gefällt **dem** Kind. (The child likes that.) |
| **gehören** | to belong to | Das Buch gehört **dem** Lehrer. (The book belongs to the teacher.) |
| **antworten** | to answer | Antworte **mir** bitte! (Answer me please!) |
| **glauben** | to believe | Ich glaube **dir**. (I believe you.) |
| **schmecken** | to taste | Die Pizza schmeckt **dem** Mann. (The pizza tastes good to the man.) |

### Part 5: Complete Summary Table
**(সম্পূর্ণ সারাংশ সারণী)**

#### Definite Articles (The - টি/টা)
| Case | Question | Masculine | Neuter | Feminine | Plural |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Nominativ** | Wer? Was? (Who? What?) | **der** | **das** | **die** | **die** |
| **Akkusativ** | Wen? Was? (Whom? What?) | **den** | **das** | **die** | **die** |
| **Dativ** | Wem? (To whom?) | **dem** | **dem** | **der** | **den** (+n) |

#### Indefinite Articles (A/An - একটি/একজন)
| Case | Masculine | Neuter | Feminine |
| :--- | :--- | :--- | :--- |
| **Nominativ** | **ein** | **ein** | **eine** |
| **Akkusativ** | **einen** | **ein** | **eine** |
| **Dativ** | **einem** | **einem** | **einer** |

#### Personal Pronouns (ব্যক্তিগত সর্বনাম)
| Nominativ | Akkusativ | Dativ |
| :--- | :--- | :--- |
| ich | **mich** (me) | **mir** (to me) |
| du | **dich** (you) | **dir** (to you) |
| er | **ihn** (him) | **ihm** (to him) |
| sie | **sie** (her) | **ihr** (to her) |
| es | **es** (it) | **ihm** (to it) |
| wir | **uns** (us) | **uns** (to us) |
| ihr | **euch** (you guys) | **euch** (to you guys) |
| sie | **sie** (them) | **ihnen** (to them) |

### Final Tips (শেষ টিপস):

1.  **Mr. MN** for Dativ: Masculine & Neuter → **dem**, Feminine → **der**, Plural → **den** + **n**
2.  **W-Fragen**: Ask **Wen?** (Akkusativ) or **Wem?** (Dativ)
3.  **Prepositions**: Memorize them in groups (pure Akk, pure Dat, mixed)
4.  **Two-way prepositions**: Movement = Akkusativ, Location = Dativ
5.  **Special verbs**: Some verbs *always* take Dativ (helfen, danken, gehören etc.)
